\doxysubsection{Detailed Description}
Representa um grafo direcionado usado para calcular o algoritmo Page\+Rank. 

A classe {\ttfamily \doxylink{classpage__rank_1_1Graph}{Graph}} permite adicionar arestas, calcular o Page\+Rank de cada nó, e imprimir o grafo. Ela utiliza uma lista de adjacência para representar o grafo.
