\doxysubsection{Detailed Description}
Represents a directed graph used to compute the Page\+Rank algorithm. 

The {\ttfamily \doxylink{classpage__rank_1_1Graph}{Graph}} class allows adding edges, calculating the Page\+Rank of each node, and printing the graph. It uses an adjacency list to represent the graph.
