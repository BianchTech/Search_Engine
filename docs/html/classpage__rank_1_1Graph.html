Detailed Description

Representa um grafo direcionado usado para calcular o algoritmo PageRank.

A classe Graph permite adicionar arestas, calcular o PageRank de cada nó, e imprimir o grafo. Ela utiliza uma lista de adjacência para representar o grafo.

Calcula o PageRank de todos os vértices do grafo.

Esta função utiliza o algoritmo PageRank para calcular a pontuação de cada vértice no grafo, iterando até o número máximo de iterações ou até que a variação seja menor que a tolerância especificada.

Parameters
damping_factorFator de amortecimento utilizado no cálculo do PageRank. Valor padrão é 0.85.
max_iterationsNúmero máximo de iterações para o algoritmo. Valor padrão é 100.
tolTolerância para a convergência do algoritmo. Valor padrão é 1e-6.
on_printFlag para controlar se as iterações devem ser impressas. Valor padrão é false.
Returns
Mapa contendo o PageRank de cada vértice, onde a chave é o vértice e o valor é o PageRank.
